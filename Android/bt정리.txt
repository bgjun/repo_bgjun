BTMsg bundle data에 들어가는 항목
1) SLED_BT_DEVICE_FOUND - startscan시에 오는 callback
b.putString(ConstantsBT.BT_BUNDLE_NAME_KEY, device.getName());
b.putString(ConstantsBT.BT_BUNDLE_ADDR_KEY, device.getAddress());
b.putInt(ConstantsBT.BT_BUNDLE_BOND_STATE_KEY, device.getBondState());

SLED_BT_DEVICE_FOUND
이 callback message는 ~~할때 전달된다.
이 message는 bundle 을 항상 포함한다.
bundle은 xxx yyy zzz 을 갖고 있다.
이 값을 얻어오는 키는 xxx yyy zzz이다.
xxx로 얻어 올수 있는 data는 string 형식이다.
yyy로 얻어 올 수 있는 data는 int 형식이다.
zzz로 얻어 올수 있는 data는 string 형식이다.

**아래내용은 공통 putString 과 putInt
ConstantsBT.BT_BUNDLE_NAME_KEY의 경우 putString으로 data를 넣으니 user 입장에서는 String으로 받아야하고
ConstantsBT.BT_BUNDLE_BOND_STATE_KEY의 경우 putint로 data를 넣으니 user 입장에서는 int로 받아야함


2) SLED_BT_BOND_STATE_CHAGNED - pairing 상태 변경시에 오는 callback
b.putString(ConstantsBT.BT_BUNDLE_NAME_KEY, device.getName());
b.putString(ConstantsBT.BT_BUNDLE_ADDR_KEY, device.getAddress());
b.putInt(ConstantsBT.BT_BUNDLE_BOND_STATE_KEY, device.getBondState());
b.putInt(ConstantsBT.BT_BUNDLE_BOND_NEW_STATE_KEY, newtState);
b.putInt(ConstantsBT.BT_BUNDLE_BOND_PREV_STATE_KEY, previousState);

3) SLED_BT_ACL_CONNECTED - connected 상태로 변경시에 오는 callback
b.putString(ConstantsBT.BT_BUNDLE_NAME_KEY, device.getName());
b.putString(ConstantsBT.BT_BUNDLE_ADDR_KEY, device.getAddress());
b.putInt(ConstantsBT.BT_BUNDLE_BOND_STATE_KEY, device.getBondState());

4) SLED_BT_ACL_DISCONNECT_REQUESTED - disconnect request가 오면 전달되는 callback
b.putString(ConstantsBT.BT_BUNDLE_NAME_KEY, device.getName());
b.putString(ConstantsBT.BT_BUNDLE_ADDR_KEY, device.getAddress());
b.putInt(ConstantsBT.BT_BUNDLE_BOND_STATE_KEY, device.getBondState());

5) SLED_BT_ACL_DISCONNECTED - disconnected 상태로 변경시에 오는 callback
                b.putString(ConstantsBT.BT_BUNDLE_NAME_KEY, device.getName());
                b.putString(ConstantsBT.BT_BUNDLE_ADDR_KEY, device.getAddress());
                b.putInt(ConstantsBT.BT_BUNDLE_BOND_STATE_KEY, device.getBondState());

                
                
6)SLED_BT_STATE_CHANGED - 단말의 bt 상태가 변경되면 오는 callback ex>on /off
                b.putInt(ConstantsBT.BT_BUNDLE_BOND_NEW_STATE_KEY, newtState);
                b.putInt(ConstantsBT.BT_BUNDLE_BOND_PREV_STATE_KEY, previousState);
                
7)SLED_BT_ADAPTER_CONNECTION_STATE_CHANGED - 항목 언급 안해도됨
                b.putString(ConstantsBT.BT_BUNDLE_NAME_KEY, device.getName());
                b.putString(ConstantsBT.BT_BUNDLE_ADDR_KEY, device.getAddress());
                b.putInt(ConstantsBT.BT_BUNDLE_BOND_STATE_KEY, device.getBondState());
                b.putInt(ConstantsBT.BT_BUNDLE_CON_NEW_STATE_KEY, newtState);

메시지만 전달
10) SLED_BT_PAIRING_REQUEST - pair request가 올경우 전달되는 callback
메시지만 전달
